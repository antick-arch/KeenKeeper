# KeenKeeper

A modern web application designed to help you maintain and strengthen your meaningful relationships by tracking interactions and setting contact goals with friends.

## Description

KeenKeeper is a friendship management application that helps you never lose touch with the people who matter. Set contact goals for each friend, track when you last reached out, and get insights on your relationship status. Never miss an opportunity to reconnect with someone important to you.

## Technologies Used

- **Next.js 14+** - React framework for production
- **React** - UI library
- **Tailwind CSS** - Utility-first CSS framework
- **React Icons** - Icon library
- **JavaScript** - Programming language
- **JSON** - Data storage

## Key Features

### 1. **Friend Management Dashboard**
   - View all your friends in an organized grid layout
   - Track relationship status (On Track, Almost Due, Overdue)
   - See friend details with photos, bios, and contact preferences
   - Filter and organize friends by various criteria

### 2. **Interaction Tracking**
   - Log quick check-ins via Call, Text, or Video
   - Automatic tracking of days since last contact
   - Set personalized contact goals for each friend (e.g., connect every 14 days)
   - View next due date for planned interactions

### 3. **Analytics & Insights**
   - Dashboard statistics showing total friends, on-track relationships, and those needing attention
   - Real-time calculations based on your JSON data
   - Visual representation of friendship health
   - Monthly interaction tracking
